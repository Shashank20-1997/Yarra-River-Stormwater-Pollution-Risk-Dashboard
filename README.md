# Yarra River Stormwater Pollution Risk Dashboard
![Dashboard](Dashboard_Screenshot.png)

## Project Overview
This Tableau dashboard analyses stormwater pollution risk in the Yarra River by examining E. coli levels, rainfall patterns, and site-level risk indicators.

## Objective
The objective was to convert raw environmental monitoring data into a clear Tableau dashboard that helps identify high-risk sites, monitor trends, and support data-driven decision-making.

## Tools Used
- Tableau
- Microsoft Excel
- Data Cleaning
- Data Visualisation

## Datasets Used
- EPA Yarra Watch E. coli water-quality data
- Bureau of Meteorology daily rainfall data

## Dashboard Features
- Total water-quality samples
- Average E. coli level
- High-risk sample count
- High-risk percentage
- Risk by monitoring site
- Rainfall vs E. coli analysis
- E. coli trend over time

## Key Insights
- 961 total water-quality samples were analysed.
- 39.33% of samples were classified as high risk.
- Kew recorded the highest average E. coli level.
- Rainfall showed a relationship with elevated E. coli readings, although site-level conditions also influenced results.

## Data Quality Note
During preparation, inconsistent site naming was identified, especially for Healesville. These values were grouped into a standard site name instead of deleting valid records.
